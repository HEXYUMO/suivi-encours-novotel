🔑 Réinitialiser le mot de passe d'un utilisateur
Firebase ne permet pas de changer le mot de passe d'un autre utilisateur depuis l'application.

Procédure si un utilisateur oublie son mot de passe :
1. Allez dans Firebase Console → Authentication → Utilisateurs
2. Cliquez sur les 3 points à droite de l'email → "Modifier l'utilisateur"
3. Saisissez directement un nouveau mot de passe → Enregistrer
4. Communiquez le nouveau mot de passe à l'utilisateur
